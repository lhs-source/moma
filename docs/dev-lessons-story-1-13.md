# Story 1.13 개발 교훈

**Story**: 주간 식재료 구매 한도 기반 요리 시뮬레이터 (2025-10-28)

---

## 🔍 **주요 실수 및 교훈**

### **1. 기존 코드 분석 부족**
- ❌ **실수**: 95% 완성된 상태인데 새 컴포넌트 생성 시도
- ✅ **교훈**: **실제 코드 검토 > Architecture 문서**

### **2. CSS 클래스 선택 오류**
- ❌ **실수**: `text-destructive` (커스텀) → 작동 안 함
- ✅ **교훈**: `text-red-600` (표준) → 안전하고 확실

### **3. 과도한 기능 구현**
- ❌ **실수**: 사용자가 원하지 않는 디버그 표시, 캐릭터별 관리 등
- ✅ **교훈**: **핵심 요구사항만 정확히 구현**

---

## 💡 **핵심 지식**

### **프로젝트 구조**
```typescript
// 이미 완성된 상태
src/views/cooking/index.vue          // 메인 UI
MaterialsSummary.vue                 // 경고 로직 존재
getBuyableCount() = NPC수 × 30개    // 주간 한도 계산
```

### **비즈니스 로직**
- **구매 가능**: NPC가 판매하는 식재료만 (후추, 설탕 등)
- **구매 불가**: 감자, 우유, 허브 등은 표시 안 함
- **경고 조건**: `필요량 > 주간한도` 시 붉은색 표시
